function coswindow (in d, in h, in n)

Detailed Description

Generates a custom cosine window with flattened sides.

Creates a 1D symmetric window of length n that transitions using a cosine shape over a width determined by d, and flattens to a base level of h at the ends.

Parameters
dWidth of cosine transition (0 < d < 0.5)
hFloor level at edges (0 < h < 1)
nTotal number of samples
Return values
val1D array of length n containing the window values
